- **Кэш-фикс:** `fetchJson` теперь `cache:"no-store"` + no-cache dev-сервер — иначе браузер отдаёт старые JSON/JS-модули (ловил на этом «1/7» вместо «1/2»). При смене порта origin получает cold cache.

## Журнал решений по ходу
- **Фаза 1:** `createBattleUnit` оставляет `attackRange` для melee = baseAttackReach+meleeBonus (падение в дефолт), для ranged — из класса (лучник 12). Врагам `attackRange` в данных волны = абсолютное значение (враж. лучник Лок.2 = 12).
- **Фаза 1:** для Фазы 1 в waves.json только 2 волны (1-1, 1-2). Победа = зачистка последней → при 2 волнах игра завершается быстро, это ожидаемо; остальные волны добавляются с механиками.
- **Кэш:** добавлен `cache:"no-store"` + `tools/nocache_server.py`; dev-хук `window.gameState` в main.js.
