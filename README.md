# Fighting Dudes Prototype

Небольшой веб-прототип мобильной idle/battle игры на `HTML + CSS + JavaScript`, рассчитанный на запуск как статический сайт и публикацию в GitHub Pages.

## Локальный запуск

Важно: проект нельзя открывать просто двойным кликом по `index.html`, потому что игровые данные из `data/*.json` загружаются через `fetch()`.

### Вариант 1. Через Python

Если установлен Python:

```bash
python -m http.server 8000
```

Потом открой:

```text
http://localhost:8000
```

### Вариант 2. Через Node.js

Если установлен Node.js:

```bash
npx serve .
```

Или:

```bash
npx http-server .
```

После запуска открой URL, который покажет команда. Обычно это что-то вроде:

```text
http://localhost:3000
```

### Вариант 3. Через VS Code

Если пользуешься VS Code:

1. Открой папку проекта.
2. Поставь расширение `Live Server`.
3. Запусти `Open with Live Server` для [index.html](D:\Git\FightingDudes\index.html).

## Структура

- [index.html](D:\Git\FightingDudes\index.html) — основной экран прототипа.
- [styles.css](D:\Git\FightingDudes\styles.css) — стили и адаптивность.
- [js/main.js](D:\Git\FightingDudes\js\main.js) — запуск игры и game loop.
- [js/game](D:\Git\FightingDudes\js\game) — состояние, фабрики, системы и UI.
- [data](D:\Git\FightingDudes\data) — JSON-настройки баланса, волн, экипировки и прогрессии.
- [architecture.md](D:\Git\FightingDudes\architecture.md) — краткий технический конспект по устройству проекта.

## Что уже есть в прототипе

- Покупка юнитов за золото.
- Merge двух одинаковых юнитов и `Mass Merge`.
- Назначение юнитов в шахты и прокачка шахт.
- Покупка экипировки через гарнизон.
- Необратимый деплой юнита в бой.
- Волны врагов, замок и победа после его уничтожения.
- Позиционное поле боя:
  - союзники появляются слева и идут вправо
  - враги появляются справа
  - юниты встают в дорожки и блокируют друг друга

## Публикация в GitHub Pages

Так как это статический сайт, для GitHub Pages достаточно залить содержимое репозитория и публиковать корень проекта.

Если `index.html` лежит в корне, дополнительная сборка не нужна.
